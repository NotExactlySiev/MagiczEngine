# MagiczEngine
Rendering engine based on OpenGL, Inspired by P.F.Magic's Petz series

Tested only using G++ compiler on MingW32 (Msys2), may not work on other platforms.

## Libraries and flags used to compile:
- -lmingw32 
- -lSDL2main 
- -lSDL2.dll 
- -luser32 
- -lgdi32 
- -lwinmm 
- -ldxguid 
- -lopengl32 
- -lglew32.dll
